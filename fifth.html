Z-index:

z-index is bascially use for gave the order to the elements.Elements with higher z-index value displays on the top of the elements.Suppose we have to visible the 4th number element on 1st number so z-index comes in the picture.In simple word it defines the order of elements.The last index by-default as highest.

Note:- z-index works only on the positioned elements like absolute,relative,fixed etc...

Example of Z-index:-

box1 (Z-index 2)
box2 (Z-index 3)
box3 (Z-index 1)
